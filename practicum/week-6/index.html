In practicum 5 you created a react application, imagequiz, and deployed it to https://your-github-username.github.io/imagequiz/. In this practicum you extend your imagequiz application to have the following functionality:
  1. When users click on an image, they are navigated to a page (this page is a page in your application) that contains a quiz. The user can then take the quiz.
  2. The file data.js contains the quizzes.
  3. Each quiz has six questions.
  4. Each question on the quiz has the following format: a picture and three options to choose from with only one of them being the correct answer. Each option contains a word supposed to be a name for the flower in the picture. The user can only select one option in each question.
  5. At the end of the quiz the score of the user will be shown on the page. The user can reapet the same quiz or can go to the home page to start another quiz.
This practicum resembles the image quiz application of Cambridge Dictionary which is available at https://dictionary.cambridge.org/us/plus/quiz/image/. Have a look at the Cambridge Dictionary image quiz application to gain a better understanding of the practicum requirements.
